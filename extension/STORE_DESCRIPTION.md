## Name
TTS Reader - Text to Speech

## Summary (132 chars max)
Read any webpage aloud with one click. Adjustable speed, voice selection, keyboard shortcuts, and highlight-as-you-read.

## Description
TTS Reader turns any webpage into an audiobook. Select text or read the full page aloud with a single click — no account required.

**Key Features:**
- One-click reading — highlight text or read entire pages
- Speed control from 0.5x to 2x for comfortable listening
- Choose from all voices available on your system
- Keyboard shortcut (Alt+R) to start and stop reading instantly
- Highlight-as-you-read so you never lose your place
- Works on any webpage — articles, docs, emails, and more

**Privacy First:**
- Uses your browser's built-in Web Speech API — no data leaves your device
- No tracking, no analytics, no data collection
- No account required to use

**Pro Features ($2/mo):**
- Custom keyboard shortcuts
- Priority voice engine selection
- Advanced reading controls

Perfect for students, researchers, accessibility needs, proofreading, or anyone who prefers listening to reading. A faster, simpler alternative to Speechify and Read Aloud.
